# 🦄 MathMagic Academy

A fun, interactive maths and science learning website for young learners (Class 2–3, ages 6–8). Unicorn/space theme with quizzes, XP points, trophies, and randomised questions.

## ✨ Features

- **725+ quiz questions** across 15 modules
- **Random question selection** — 10 new questions picked every attempt
- **Shuffled answer order** — answers in a different order each time
- **XP & Trophy system** — earn points and unlock achievements
- **Progress tracking** — per-module star ratings saved in browser
- **British curriculum aligned** — Class 2 maths, Class 3 preview, and science
- **iPad friendly** — fully responsive layout
- **Works offline** — single HTML file, no server needed

## 🚀 How to Use

**Option 1 — Open locally:** Download `index.html` and open in any browser.

**Option 2 — Host free on Netlify:** Drag `index.html` to [app.netlify.com/drop](https://app.netlify.com/drop)

**Option 3 — iPad home screen:** Open in Safari → Share → Add to Home Screen
