# 🦄 MathMagic Academy

A fun, interactive maths and science learning website built for young learners (Class 2–3, ages 6–8). Features a unicorn/space theme with quizzes, XP points, trophies, and randomised questions.

## ✨ Features

- **725+ quiz questions** across 15 modules
- **Random question selection** — 10 new questions picked every attempt
- **Shuffled answer order** — answers appear in a different order each time
- **XP & Trophy system** — earn points and unlock achievements
- **Progress tracking** — per-module star ratings saved in browser
- **British curriculum aligned** — Class 2 maths, Class 3 preview, and science
- **iPad friendly** — fully responsive layout
- **Works offline** — single HTML file, no server needed

## 🚀 How to Use

### Option 1 — Open locally
Download `index.html` and open it in any browser. No installation needed.

### Option 2 — Host online (free)
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop `index.html`
3. Get an instant live URL!

### Option 3 — Add to iPad home screen
1. Open the URL in Safari
2. Tap the Share button → **Add to Home Screen**
3. It appears as an app icon 🎉

## 🔐 Login
Default PIN: **1234**

To change it, open `index.html` in a text editor and find the `doLogin()` function.
